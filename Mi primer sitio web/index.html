El lenguaje html

Que es Html?

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían
ser párrafos, una lista con viñetas, o imágenes y tablas de datos

Para que sirve?

HTML se usa para definir la página de inicio de una página y organizar la forma en que se muestra, HTML permite enlaces a otras páginas o documentos.
